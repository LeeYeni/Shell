### **project initializer**
---
### **개요**
2025.11.20(목) ~ 2025.11.25(화)까지 진행하는 프로젝트로,
졸업 프로젝트에 앞서 수행하는 사전 스터디입니다.

이번 스터디에서는 아래의 내용을 다뤘습니다.
* 최상위 및 하위 디렉토리 구조 생성
* 특정 디렉토리 내 템플릿 파일 생성
* 특정 파일 내 텍스트 추가
---
### **Usage**
```bash
sudo apt-get install tree
bash 001_project_initializer.sh
```
